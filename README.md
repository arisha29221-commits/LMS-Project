## Description
This is a basic web-based Learning Management System developed as a BSCS 1st semester project.

## Features
- User Login System (PHP + MySQL)
- Course Display from Database
- Assignment Submission
- Quiz System with Score

## Technologies Used
- HTML, CSS, Bootstrap
- PHP
- MySQL
- XAMPP

## How to Run
1. Install XAMPP
2. Place project folder in htdocs
3. Start Apache and MySQL
4. Open browser and go to:
   http://localhost/LMS

## Note
This is a basic prototype and not a full LMS system.
